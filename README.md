# Laurel

A very unremarkable piece of forum software.

## Features

- Spartan (but responsive) design
- Markdown support
- Embeddable images
- Collapsible threads
- Linkable posts
- Voting
- An account system

What more could you ever want?

## Usage

1. `$ git clone $REPO`
2. `$ git submodule update --init`
3. `$ cd setup && ./init.sh`
4. Host the PHP site
